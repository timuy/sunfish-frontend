# sunfish-frontend
Sunfish FrontEnd

1. Clone the repo: git clone https://github.com/timuy/sunfish-frontend.git
2. Install Docker Desktop (https://www.docker.com/products/docker-desktop/)
3. cd to sunfish-backend
4. Run docker-compose to spin up Docker locally docker-compose up --build -d
5. Use IDE of your choice to connect to Docker container or via command line: docker exec -it sunfish-frontend bash

the UI is at http://localhost:3000


Please make sure that backend in the other repo (https://github.com/timuy/sunfish-backend) is up.
